# Agent Protocol Tech Tree

A visual guide to the emerging open standards for AI agents — how they talk to tools, to each other, and to businesses.

## View

Visit the live site at **https://&lt;username&gt;.github.io/agent-protocols/** (update after enabling GitHub Pages).

## Local development

Open `index.html` in a browser. No build step required — the page loads `data.yaml` at runtime via js-yaml.

## Structure

| File | Purpose |
|------|---------|
| `index.html` | Page shell |
| `style.css` | Styles |
| `app.js` | Renders the tech tree from data |
| `data.yaml` | All protocol data (titles, descriptions, relationships) |
